Projects of SBS teams is a simulation to the real life projects of labor market

Basket teck :
The concept of this project is Using a trash can for recycling and get points for this environmentally friendly behavior.

Bri (banana recycling industry) :
The concept of this project is to recycle banana waste into many things, which preserves the environment.

Gymart :
The concept of this project is based on a subscription for different gyms in one subscription, which allows you to enter more than one gym with the same subscription.

Waseela :
The concept of This project is based on programming courses for children, which allows children to study programming at a young age, which leads to the creation of a programming and digitally aware generation.

Izaar :
The concept of a project for a clothing design company specializing in supplying clothing for events and parties, especially Pharaonic clothing.
